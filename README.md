# BoilerPlate App

## Description

This is a boilerplate project file to create any react applications with authentication set up

## Getting Started

1. Copy this repo
2. 'npm install'
3. 'npm run start:dev'
4. Create your database and replace information in designated files
5. 'npm run seed'
6. replace secret key in .env
7. Have fun!

## Technologies Used

1. redux
2. react/ react-dom
3. express
4. nodemon
5. postgres
6. sequelize
7. webpack
8. babel
9. jsonwebtoken & bcrypt
